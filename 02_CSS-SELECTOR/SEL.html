CSS SELECTOR

1. Class selector
Syntax : .classname {

} (We use this 90% of the time)
2. Id selector
Syntax : #idname {

}
3. Attribute selector
Syntax : [attributename] {

}
4. Combinator selector
Syntax : dl dd {

} (Pointing specificly such as dd inside dl)
5. Hover selector
Syntax : () :hover {

} (h1:hover -> when user hover over h1, changes will be seen )
