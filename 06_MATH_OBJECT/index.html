  9. Math.SQRT3 : Returns the square root of 3

Math Object Methods

  1. Math.sqrt(x) : Returns the square root of x
  2. Math.abs(x) : Returns the absolute value of x
  3. Math.ceil(x) : Rounds x up to the nearest integer
  4. Math.floor(x) : Rounds x down to the nearest integer
  5. Math.round(x) : Rounds x to the nearest integer
  6. Math.max(x, y, z, ...) : Returns the largest of the given numbers
  7. Math.min(x, y, z, ...) : Returns the smallest of the given numbers
  8. Math.random() : Returns a random number between 0 and 1

Understand infinity and NaN

Infinity is a special value in JavaScript that represents a number that is larger than any finite number. It is the result of dividing a positive number by zero or exceeding the maximum limit of a number.

NaN (Not a Number) is a special value in JavaScript that represents a value that is not a valid number. It is the result of an invalid mathematical operation, such as dividing zero by zero or trying to parse a non-numeric string as a number.

For example:

1 / 0 = Infinity

0 / 0 = NaN

Truthy and Falsy Values in JS

In JavaScript, truthy values are values that evaluate to true in a boolean context, while falsy values are values that evaluate to false. The following values are considered falsy:

  • false
  • 0
  • -0
  • NaN
  • null
  • undefined
  • "" (empty string)
