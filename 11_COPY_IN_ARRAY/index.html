Copy in Array and Objects

Shallow copy

Shallow copy is a bitwise copy of an object. It means that the new object is created with the same properties as the original object, but the properties are not copied by reference. Instead, they are copied by value. This means that if you change a property in the new object, it will not affect the original object.

Ways to copy

  • Spread operator (...)
  • Array.from()
  • slice()

Deep copy

Deep copy is a bitwise copy of an object. It means that the new object is created with the same properties as the original object, but the properties are copied by reference. This means that if you change a property in the new object, it will also affect the original object.

  • Lodash cloneDeep()
