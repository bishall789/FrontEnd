GRID

Grid and Flexbox both are parent controlled property , in which elements are controlled by parent.

In Grid we use 2 property to specify grid rows and columns given below and each elements insdie grid will occupy each grid selve.

  1. grid-template-rows: repeat(3,150px); -> This will make 3 rows with 150px .
  2. grid-template-columns: 150px 150px 150px 150px 150px; -> This will make 5 columns with 150px.

Other ways to give grid-template-columns and grid-template-rows are : using % , fr , autofill.

  1. grid-template-columns:repeat(3,30%); -> This will make 3 columns with 30% of width out of 100% width, similary for rows also.
  2. grid-template-columns:repeat(3,1fr); -> This will make 3 columns dividing total width into equal 3 sections.
  3. grid-template-columns:repeat(auto-fill, 100px); -> This will divide entire width into 100px column how much width is available.
  4. grid-template-columns:repeat(auto-fill, min-max(120px , 1fr)); -> This will give min of 120px and max of 1fr to every element on based of available width.

When we do column then it pushes other elements , so whats the solution :

There are 2 ways to solve it:

  • Either use grid-auto-flow:row dense in parent.
  • Or use grid-row-start in child with grid-column-start to ensure elements are not pushed.
