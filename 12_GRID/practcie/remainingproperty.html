  3. grid-auto-columns -> By default its auto

grid-template-areas

In grid-template-areas -> We can define rows and columns with name instead of numbers.

Previously we used to give star and end of rows and columns but in gris-template-areas -> we can use shortcut.

grid-row:Banglore -> This will be applicable for both start and end similarly we can do for grid-column.

More shortcut property -> grid-area:Banglore , this will applicable for both rows and columns start and end.

Meaning it will coevr every grid sleve where there is banglore.

Note : It will only works when its forms a square or rectangle.
