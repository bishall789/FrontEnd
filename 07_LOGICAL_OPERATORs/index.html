Logical Operators in JS

Logical AND (&&)

Logical AND (&&) returns true if both operands are true.

Example: 5 > 3 && 8 > 5

Logical OR (||)

Logical OR (||) returns true if at least one operand is true.

Example: 5 > 3 || 8 < 5

Logical NOT (!)

Logical NOT (!) returns true if the operand is false.

Example: !(5 > 3)
