MEDIA

In CSS, media helps us apply different styles based on the type of device or screen size. This is useful when designing responsive websites that work well on mobiles, tablets, laptops, and even for printing documents.

Types of Media :

  1. Screen :This is used for screens like mobiles, tablets, laptops, and desktops. It allows us to change styles based on the screen size, resolution, or orientation (portrait/landscape).
  2. Print : This is used when a webpage is printed on paper. It removes unnecessary elements (like menus or ads) and adjusts fonts for better readability.
  3. All : Combination of both

In media we can apply media as

@media print {

.hello {

color:red;

}

}

This will change color to red of .square when printing

Mostly Used Media Feautures and Queries
