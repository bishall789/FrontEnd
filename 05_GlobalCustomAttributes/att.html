HTML attributes

Attribute : HTML attributes provide additional information about HTML elements.

Global and Custom Attribute

HTML Standard : These attributes are predefined where we can use them , as src and alt are used in img , They dont make any sense in p or h

Some Standard Html tags are :

  • src and alt for img
  • href for anchor tag

Global Attribute : The global attributes are attributes that can be used with all HTML elements.

Some of the widely used Global Attributes are :

  • Hidden : hides the content , its only hidden nothing else
  • Title : shows content whenever we hover over them , content text to show when ever hover
  • Style : used to add css to the element
  • Class : used to add class to the element (CSS)
  • Id : used to add id to the element (CSS)
  • Data : used in js (JS)

Custom Attribute : Those attribute which we create as our need

<> Such as h hi="Keseho" here hi is valid custom attribute
