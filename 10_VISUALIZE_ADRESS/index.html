Non Primitive Data Types

There is 1 types of NPDT :
  1. Object

Obejct is divided into 3 categories :

  1. Object Literal
  2. Arrays
  3. Functions

Object Literal

Object Literal is a collection of key-value pairs.

Syntax :

var objectName = { key1 : value1, key2 : value2, key3 : value3 }

Example :

var person = { name : "John", age : 30, city : "New York" }

To access the value of a key, we can use dot notation or bracket notation.

Example :

person.name // John

person["age"] // 30

To add a new key-value pair, we can use dot notation or bracket notation.

Example :

person.country = "USA"

person

To delete a key-value pair, we can use the delete operator.

Example :

delete person.age

person

To check if a key exists, we can use the in operator.

Example :

"name" in person // true

"age" in person // false

To get the keys of an object, we can use the Object.keys() method.

Example :

Object.keys(person) // ["name", "city", "country"]

To get the values of an object, we can use the Object.values() method.

Example :

Object.values(person) // ["John", "New York", "USA"]

Objectseal() vs Objectfreeze()

Objectseal() :

It prevents the addition or deletion of properties from an object.

It allows the modification of existing properties.

Example :

var person = { name : "John", age : 30, city : "New York" }

Object.seal(person)

person.name = "Jane"

person.age = 25

person.country = "USA"

person // { name : "Jane", age : 25, city : "New York" }

Objectfreeze() :

It prevents the addition, deletion or modification of properties from an object.

Example :

var person = { name : "John", age : 30, city : "New York" }

Object.freeze(person)

person.name = "Jane"

person.age = 25

person.country = "USA"

person // { name : "John", age : 30, city : "New York" }

Object.seal() vs Object.freeze() :

Object.seal() allows the modification of existing properties, while Object.freeze() does not.

Object.seal() prevents the addition or deletion of properties, while Object.freeze() prevents the addition, deletion or modification of properties.

Array

An array is a collection of elements.

Syntax :

var arrayName = [ element1, element2, element3 ]

Example :

var fruits = [ "apple", "banana", "orange" ]

To access an element of an array, we can use the index of the element.

Example :

fruits[0] // "apple"

fruits[1] // "banana"

To add a new element to an array, we can use the push() method.

Example :

fruits.push("grape")

fruits // [ "apple", "banana", "orange", "grape" ]

To remove the last element of an array, we can use the pop() method.

Example :

fruits.pop()

fruits // [ "apple", "banana", "orange" ]

To remove the first element of an array, we can use the shift() method.

Example :

fruits.shift()

fruits // [ "banana", "orange" ]

To add a new element to the beginning of an array, we can use the unshift() method.

Example :

fruits.unshift("apple")

fruits // [ "apple", "banana", "orange" ]

To remove an element from a specific index, we can use the splice() method.

Example :

fruits.splice(1, 1)

fruits // [ "apple", "orange" ]

To get the length of an array, we can use the length property.

Example :

fruits.length // 2

To check if an element exists in an array, we can use the indexOf() method.

Example :

fruits.indexOf("banana") // -1

fruits.indexOf("apple") // 0

To get the last element of an array, we can use the length property.

Example :

fruits[fruits.length - 1] // "orange"

To get the first element of an array, we can use the index 0.

Example :

fruits[0] // "apple"
