Box Model CSS

Some basic rules :

  1. width and height are dependent on each other.
  2. To prevent overflow -> Avoid width and use max-width
  3. To prevent overflow -> Avoid height and use min-height
  4. Standard Way to use width and width is to use in percentage(%),
  5. When we give width 50% -> then it takes 50% of its parent width
  6. When we give height 50% -> then it takes 50% of its parent but only when its parent height is in absloute value (in px)
